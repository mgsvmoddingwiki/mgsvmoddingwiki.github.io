Custom / Addon Side-Ops / Quests created for Metal Gear Solid V: The
Phantom Pain.

See [MissionCodes\#Side_Op_ids](/MissionCodes#Side_Op_ids "wikilink")
for vanilla questIds.

[Infinite Heaven](https://www.nexusmods.com/metalgearsolidvtpp/mods/45)
is required to add custom side-ops to the game.

Note for side-op authors: Due to how the engine uses questIds must be
unique for each sideop. So if you are creating sideops you should make
sure you don't conflict with existing ones, and update this page with
the ids you intend to use.
